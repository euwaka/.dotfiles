# .dotfiles
The ugliest and best dotfiles that nobody should use.

# Requirements

Ensure you have the following installed:
- i3lock
- scrut
- tmux
- kitty
- neovim
- i3wm
- ohmyposh
- stow
- feh
- PulseAudio
- drun
- GromitMPX
- desire to fix bugs on your system (e.g., installing requirements I haven't mentioned above)
